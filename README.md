# Refold AI GTM System — Public Dashboard

This repo hosts the public mirror of the Refold AI GTM System dashboard.
It's auto-synced from the private `archit127/GTM_AI_System_Refold` repo
on every cron run (4×/day) plus on every push.

**Live URL:** https://archit127.github.io/refold-gtm-dashboard/

The dashboard reads `dashboard.json` (regenerated by the cron) and writes
team feedback directly to Supabase via the anon key (RLS-protected).
